# Study ETC

- 이 레포지토리는 몇몇의 작은 주제들을 공부하고 구현한 사항을 정리한 레포지토리입니다.
- This repository is a collection of small topics that I studied and implemented.

<br />

## 사용한 기술 (Tech Stack)

1. Express.js, Docker, Docker Compose

   - 이 프로젝트는 두 개의 간단한 Express.js 서버를 만들고 Docker 이미지를 생성한 후 두 개의 이미지를 Docker Compose로 실행시킨 프로젝트입니다.
   - This project involves creating two simple Express.js servers, creating Docker images, and running the two images using Docker Compose.

<br />

2. Express.js, Swagger UI (swagger-jsdoc, swagger-ui-express), OpenAPI Specification

   - 이 프로젝트는 CRUD 컨트롤러를 포함한 간단한 Express.js 서버를 만들고 OpenAPI Specification을 작성한 후, 그를 활용하여 Swagger UI를 생성한 프로젝트입니다.
   - This project involves creating a simple Express.js server with CRUD controllers, writing an OpenAPI Specification, and utilizing it to generate Swagger UI.

<br />

3. Node.js (npx), Docker (bat file), OpenAPI Generator

   - 이 프로젝트는 OpenAPI generator를 활용하여 두 번째 프로젝트에서 생성한 OpenAPI Specification을 프론트엔드에서 활용할 수 있도록 axios factory로 변환한 프로젝트입니다.
   - This project uses the OpenAPI generator to convert the OpenAPI Specification generated in the second project into an axios factory, allowing it to be utilized in the frontend.

<br />

4. Next.js, next-i18next

   - 이 프로젝트는 next.js에서 다국어를 위해 흔히 사용되는 next-i18next 모듈을 사용하여 다국어 표현을 구현해본 프로젝트입니다.
   - This project is an implementation of multilingual support using next-i18next which is a commonly used module in Next.js.
